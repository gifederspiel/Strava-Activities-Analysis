# Strava Activities Analysis

Visualise your Strava running history — pace trends, weekly volume, heart rate analysis, and pace improvement over time.

## Setup

```bash
pip install -r requirements.txt
streamlit run app.py
```

Then open [http://localhost:8501](http://localhost:8501) in your browser.

## Usage

Upload your `activities.csv` from your Strava data export.  
An example file is included so you can try it out right away.

**How to export from Strava:** Settings → My Account → Download or Delete Your Account → Request Your Archive. Open the ZIP and upload the `activities.csv` inside.
